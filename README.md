## Getting Started

### Prerequisites

1. npm
  ```bash
  npm i -g npm@latest
  ```
2. redis-server
3. ffmpeg
4. mysql
5. .env
6. yarn

### Installation

1. Clone the repo
  ```bash
  git clone https://github.com/sleepy-wood/server-api.git
  ```
2. Install NPM packages
  ```bash
  yarn
  ```
3. Run App
  ```bash
  yarn start:dev
  ```
